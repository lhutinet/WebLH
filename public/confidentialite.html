Politique de confidentialité

Le site digitalh.fr respecte votre vie privée. Cette politique de confidentialité explique comment vos données sont utilisées.

1. Données collectées

Aucune donnée personnelle n’est collectée directement via ce site (pas de formulaires, pas de compte utilisateur).

2. Utilisation de Google Analytics

Ce site utilise Google Analytics, un service d’analyse fourni par Google, pour mesurer l’audience et améliorer votre expérience.

  • Les données collectées incluent votre adresse IP, type d’appareil, pages visitées, durée de visite, etc.
  • L'adresse IP est partiellement anonymisée (fonction d'anonymisation activée).

Ces données sont traitées uniquement à des fins statistiques.

3. Cookies

Des cookies sont utilisés uniquement pour les statistiques via Google Analytics. Vous pouvez choisir d’accepter ou de refuser ces cookies lors de votre première visite via le bandeau d’information.
